# AI Image Prompts for Branding Assets

Use these curated prompts in AI generators (e.g., Midjourney, DALL-E 3, Stable Diffusion XL) to generate assets for your portfolio banner backgrounds.

---

## 1. Profile Banners (Aspect Ratio: 3:1)

### **Prompt: Minimalist Tech Grid (Midjourney / SDXL)**
> `/imagine prompt: Clean minimalist abstract background banner, slate gray and deep royal blue gradients, subtle glowing ethernet nodes, digital lines, high tech circuit board vector style, depth of field, high contrast, clean vectors, 8k resolution, aspect ratio 3:1 --v 6.0`

### **Prompt: Neural Network Connections (DALL-E 3)**
> `Create a minimalist banner with aspect ratio 3:1. The background is dark charcoal grey. On the right, show a beautiful, clean, glowing light-blue neural network cluster with lines and glowing circles. On the left, it should fade to a solid dark grey. No text.`

## 2. Repository Banners & Covers

### **Prompt: Object Detection / Computer Vision Theme (Midjourney)**
> `/imagine prompt: Dark industrial warehouse, neon green and cyan bounding boxes highlighting tools and machinery, computer vision overlay interface, futuristic, cybernetic, cinematic lighting, flat vector graphic look, aspect ratio 16:9 --v 6.0`

### **Prompt: Generative AI / Agent Matrix (SDXL)**
> `/imagine prompt: A clean diagram of digital brains, nodes transmitting code pulses, cool blue and emerald green accents, minimalist white background, soft ambient occlusion, isometric view, aspect ratio 16:9`
