# Security Policy

## Supported Versions

Only the latest version of GitHub Portfolio System is currently supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| < 1.0   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please open a draft security advisory on GitHub or email the maintainer. We will address all security vulnerabilities promptly.
