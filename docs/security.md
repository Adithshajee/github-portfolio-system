# Security Policy

Please read our security guidelines and standard procedures.

## Reporting a Vulnerability

Please do **NOT** open a public issue for security findings. Instead:
- Open a private advisory on GitHub Security panel.
- Or email `adideva12345@gmail.com` with `[GPS Security]` prefix.

We will respond within 48 hours and release fixes within 7 days.

See [SECURITY.md](https://github.com/Adithshajee/github-portfolio-system/blob/main/SECURITY.md) for more details.
